Avaliação Prática - API Spring

Orientação

Estes materiais foram preparados para consulta durante a avaliação prática. Eles explicam conceitos, estruturas e exemplos gerais de Spring Boot, API REST, JPA, validações, Swagger e Git.

Atenção: durante a avaliação, consulte apenas estes arquivos. Não é permitido pesquisar em sites, usar IA, fóruns, vídeos ou projetos externos.

Manuais

01 - Como rodar o projeto

Passos básicos para abrir, configurar e executar o projeto.

02 - Estrutura Spring Boot

Explicação das camadas Model, Repository, Service, Controller e DTO.

03 - Guia CRUD

Exemplos gerais de rotas para cadastrar, listar, buscar, atualizar e remover.

04 - JPA e Relacionamentos

Resumo sobre entidades, chaves primárias e relacionamentos.

05 - Validações

Anotações para validar dados recebidos pela API.

06 - Swagger/OpenAPI

Como documentar entidades e rotas da API.

07 - Filtros

Exemplos simples de consultas personalizadas.

08 - Erros comuns

Problemas frequentes em projetos Spring Boot e como interpretar.

09 - Comandos Git

Comandos básicos para enviar o projeto ao GitHub.

10 - Checklist de entrega

Lista final para conferir antes de entregar a avaliação.
