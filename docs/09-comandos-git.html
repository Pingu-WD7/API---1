09 - Comandos Git

Voltar ao início

Verificar status do projeto

git status

Adicionar arquivos

git add .

Criar commit

git commit -m "Finaliza avaliacao pratica"

Conectar com repositório remoto

git remote add origin URL_DO_REPOSITORIO

Enviar para o GitHub

git push -u origin main

Caso a branch seja master

git push -u origin master

Verificar repositório remoto

git remote -v

Atualizar envio após novos commits

git add .
git commit -m "Ajustes finais"
git push

Cuidados na entrega

  • O repositório deve ser público.
  • O projeto deve estar descompactado.
  • Não envie apenas um arquivo .zip.
  • Confirme se o README aparece no GitHub.
  • Confirme se a pasta src aparece corretamente.
  • Confirme se a pasta docs foi enviada.
