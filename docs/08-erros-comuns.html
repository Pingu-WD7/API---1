08 - Erros comuns

Voltar ao início

Erro de banco de dados

Failed to configure a DataSource: 'url' attribute is not specified
Verifique se o arquivo application.properties possui a URL do banco, usuário, senha e driver. 
spring.datasource.url=jdbc:postgresql://localhost:5432/nome_do_banco
spring.datasource.username=postgres
spring.datasource.password=sua_senha
spring.datasource.driver-class-name=org.postgresql.Driver

Erro 400 Bad Request

A requisição foi enviada com dados inválidos.
Verifique se o JSON está correto e se o cabeçalho Content-Type está como application/json. 
{
  "nome": "Exemplo",
  "preco": 100.0
}

Erro 404 Not Found

O recurso solicitado não foi encontrado.
Verifique se o idLocal existe no banco de dados e se a rota está correta.

Erro de relacionamento nulo

DataIntegrityViolationException ou erro de chave estrangeira.
Verifique se a entidade relacionada foi buscada no banco antes de salvar. 
Categoria categoria = categoriaRepository.findById(dto.getCategoriaId())
        .orElseThrow(() -> new RuntimeException("Categoria não encontrada"));

Erro no Swagger

Swagger não abre ou retorna erro.
Verifique se a dependência springdoc-openapi está no pom.xml e se o projeto iniciou corretamente.

Erro de importação

Cannot resolve symbol
Verifique se a classe existe, se o pacote está correto e se a dependência foi carregada pelo Maven.

Erro de anotação esquecida

Algumas classes precisam de anotações para funcionar corretamente. 

@Entity
@Service
@RestController
@RequestMapping
@Id
@GeneratedValue

Erro ao atualizar

O sistema cria um novo registro em vez de atualizar.
No método de atualização, busque o registro existente, altere os campos e depois salve. 
Produto produto = buscarPorId(idLocal);
produto.setNome(dto.getNome());
produto.setPreco(dto.getPreco());
return repository.save(produto);
