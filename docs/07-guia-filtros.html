07 - Filtros

Voltar ao início

O que são filtros?

Filtros permitem buscar registros de acordo com alguma informação. Por exemplo: buscar produtos pelo nome, buscar clientes por cidade ou buscar eventos por categoria.

Exemplo de filtro por nome

No repository, podemos criar métodos usando o padrão do Spring Data JPA. 

public interface ProdutoRepository extends JpaRepository<Produto, Long> {

    List<Produto> findByNomeContainingIgnoreCase(String nome);

}

Service usando filtro

@Service
public class ProdutoService {

    @Autowired
    private ProdutoRepository repository;

    public List<Produto> filtrarPorNome(String nome) {
        return repository.findByNomeContainingIgnoreCase(nome);
    }
}

Controller usando @RequestParam

@GetMapping("/filtro")
public List<Produto> filtrarPorNome(@RequestParam String nome) {
    return service.filtrarPorNome(nome);
}

Exemplo de chamada

GET /produtos/filtro?nome=mouse

Consulta por entidade relacionada

Também é possível buscar registros relacionados a outro registro. 

List<Produto> findByCategoriaId(Long categoriaId);

Exemplo de rota:

GET /produtos/categoria/1

Controller com PathVariable

@GetMapping("/categoria/{idCategoria}")
public List<Produto> listarPorCategoria(@PathVariable Long idCategoria) {
    return service.listarPorCategoria(idCategoria);
}
