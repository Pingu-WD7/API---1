10 - Checklist de entrega

Voltar ao início

Antes de entregar, confira:

☐ O projeto abre corretamente na IDE.
☐ O projeto roda sem erro.
☐ O banco PostgreSQL foi criado.
☐ O application.properties está configurado.
☐ As entidades possuem @Entity.
☐ Os atributos id possuem @Id e @GeneratedValue.
☐ Os relacionamentos foram configurados corretamente.
☐ Os repositories estendem JpaRepository.
☐ Os services possuem as regras necessárias.
☐ Os controllers possuem as rotas solicitadas.
☐ Os DTOs estão sendo usados na entrada e/ou saída de dados.
☐ As validações foram aplicadas.
☐ O Swagger abre corretamente.
☐ As rotas aparecem no Swagger.
☐ É possível cadastrar locais.
☐ É possível cadastrar eventos vinculados a locais.
☐ É possível cadastrar inscrições vinculadas a eventos.
☐ A listagem de inscrições por evento funciona.
☐ O filtro por nome do evento funciona.
☐ A listagem de eventos por local funciona.
☐ O projeto foi enviado para um repositório público no GitHub.
☐ O link do repositório foi entregue conforme orientação do professor.
