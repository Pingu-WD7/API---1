O que é Swagger?

Swagger é uma ferramenta que gera uma documentação visual para testar e visualizar as rotas da API.

Dependência no pom.xml

<dependency>
    <groupId>org.springdoc</groupId>
    <artifactId>springdoc-openapi-starter-webmvc-ui</artifactId>
    <version>2.8.9</version>
</dependency>

Acessando o Swagger

http://localhost:8080/swagger-ui.html

ou:

http://localhost:8080/swagger-ui/index.html

@Tag

Usada para organizar os controllers na documentação. 

@Tag(name = "Produtos", description = "Rotas para gerenciamento de produtos")
@RestController
@RequestMapping("/produtos")
public class ProdutoController {
}

@Operation

Usada para descrever uma rota. 

@Operation(summary = "Lista todos os produtos")
@GetMapping
public List<Produto> listar() {
    return service.listarTodos();
}
