O que é CRUD?

CRUD é um conjunto básico de operações usado em sistemas:

Letra Significado Exemplo
C Create Cadastrar
R Read Listar ou buscar
U Update Atualizar
D Delete Remover

Rotas comuns

Método HTTP Rota Função
GET /produtos Lista todos os registros
GET /produtos/{idLocal} Busca um registro por idLocal
POST /produtos Cadastra um novo registro
PUT /produtos/{idLocal} Atualiza um registro existente
DELETE /produtos/{idLocal} Remove um registro

Exemplo de controller CRUD

@RestController
@RequestMapping("/produtos")
public class ProdutoController {

    @Autowired
    private ProdutoService service;

    @GetMapping
    public List<Produto> listar() {
        return service.listarTodos();
    }

    @GetMapping("/{idLocal}")
    public Produto buscarPorId(@PathVariable Long idLocal) {
        return service.buscarPorId(idLocal);
    }

    @PostMapping
    public Produto cadastrar(@RequestBody Produto produto) {
        return service.salvar(produto);
    }

    @PutMapping("/{idLocal}")
    public Produto atualizar(@PathVariable Long idLocal, @RequestBody Produto produto) {
        return service.atualizar(idLocal, produto);
    }

    @DeleteMapping("/{idLocal}")
    public void deletar(@PathVariable Long idLocal) {
        service.deletar(idLocal);
    }
}
