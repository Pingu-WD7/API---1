01 - Como rodar o projeto

Voltar ao início

1. Abrir o projeto

Abra o projeto na IDE utilizada em aula. Aguarde a IDE carregar as dependências do Maven.

2. Verificar o arquivo pom.xml

O arquivo pom.xml contém as dependências do projeto, como Spring Web, Spring Data JPA, PostgreSQL, Validation e Swagger/OpenAPI.

3. Criar o banco de dados no PostgreSQL

Antes de rodar a aplicação, crie um banco de dados no PostgreSQL. Pode ser usado o pgAdmin ou o comando: 

CREATE DATABASE ___________;

4. Configurar o application.properties

No arquivo de propriedades da aplicação, verifique a conexão com o banco. 


spring.datasource.url=jdbc:postgresql://localhost:5432/nomeDoBanco
spring.datasource.username=
spring.datasource.password=


spring.jpa.hibernate.ddl-auto=update
spring.jpa.show-sql=true
spring.jpa.properties.hibernate.format_sql=true

5. Rodar a aplicação

Execute a classe principal do projeto

6. Testar no navegador

Depois que o projeto iniciar, teste o Swagger: 

http://localhost:8080/swagger-ui.html

ou:

http://localhost:8080/swagger-ui/index.html

7. Testar no Postman

Você também pode testar as rotas usando o Postman. Lembre-se de enviar dados em JSON e usar o cabeçalho: 

Content-Type: application/json
